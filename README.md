# Student Academic Portal

## Project Description
This project is a Python-based Student Academic Portal developed using Tkinter. It manages student registration, module score entry, and automatic calculation of grades, GPA, and CGPA. The system generates detailed academic reports and supports efficient student data management.

## Features
- Student registration system
- Module selection and score input
- Automatic grade calculation
- GPA and CGPA computation
- Academic report generation
- Record editing and deletion
- Search and filtering system
- Statistical dashboard

## Technologies Used
- Python
- Tkinter GUI
- Statistics module
- UUID module

## How to Run
1. Install Python 3
2. Run `main.py`
3. Use the graphical interface to manage student records
